== Description ==

TLead Stream WordPress plugin to track leads with integration to Gravity Forms and Twillio. Can be used to send whisper messages while using 
Twilio for call routing/forwarding.

== Installation ==

Here is how to get the WP Lead Stream plugin working with your site.

1. Upload `wp-lead-stream` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin settings as you would like
4. Configure your Twilio Webhooks settings using the url at the bottom of the settings page.
5. If you want to track web form submissions use Gravity forms plugin and configure the WPLeadStream section of the form settings.
